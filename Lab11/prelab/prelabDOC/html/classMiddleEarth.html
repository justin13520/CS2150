This method returns the distance between the two passed cities.

If we assume that the hash table (i.e. the map) is O(1), then this method call is also O(1). This can also be used to find the weight/cost of each path between cities.

vector< string > MiddleEarth::getItinerary ( unsigned int  length)

Gets a vector of strings that represents the cities travelled to.

The first city is the original start point as well as the end point. The number of cities passed in does not include this start/end point (so there will be length+1 entries in the returned vector).

Returns
Returns the list of cities to travel to.
